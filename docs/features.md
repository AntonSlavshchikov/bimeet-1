# MeetUp — Design Rationale

## Концепция: "Warm Indigo"

Приложение MeetUp помогает людям организовывать встречи — от дня рождения до корпоративного созвона. Дизайн должен ощущаться **живым и тёплым**, но при этом **профессиональным и современным**.

---

## Цвета

### Почему Indigo, а не Purple?

Исходный бренд-цвет — фиолетовый (`#7c3aed`). Проблема: это один из самых используемых «tech-purple» цветов в 2023–2025. Мы меняем его на **Indigo** (`#4F46E5`):

- Indigo — более самобытный, ассоциируется с ночью, событиями, энергией
- Меньше «generic SaaS», больше character
- Единый бренд: уходим от двоевластия purple/blue → один `#4F46E5` везде

```
Brand (Indigo):
  50:  #EEF2FF
  100: #E0E7FF
  200: #C7D2FE
  300: #A5B4FC
  400: #818CF8
  500: #6366F1
  600: #4F46E5  ← primary action
  700: #4338CA  ← hover
  800: #3730A3  ← active/pressed
  900: #312E81
```

### Фон: тёплый, а не холодный

Было: `#F5F5F7` — холодный Apple-like серый
Стало: `#F8F7F4` — тёплый off-white, ближе к бумаге

Карточки на тёплом фоне выглядят уютнее. Это важно для приложения **про людей и встречи** — пусть чувствуется камерность, а не стерильность.

### Типографика

Текстовые цвета переходят с generic `gray.*` на slate-шкалу:

| Уровень | Было | Стало |
|---------|------|-------|
| Заголовки | `gray.900` | `#0F172A` (slate-900 — глубокий, не резкий) |
| Основной текст | `gray.800` | `#0F172A` |
| Вторичный | `gray.500` / `gray.600` | `#64748B` (slate-500) |
| Третичный / placeholder | `gray.400` | `#94A3B8` (slate-400) |

Slate — более тёплый и изысканный, чем нейтральный gray.

## Градиенты карточек событий

Старые градиенты были слишком «радужными» и разнородными. Новые:

| # | Название | Цвета | Контекст |
|---|----------|-------|----------|
| 1 | Indigo → Violet | `#4F46E5 → #7C3AED` | Основной, праздничный |
| 2 | Rose → Fuchsia | `#F43F5E → #A855F7` | Романтический, вечеринка |
| 3 | Cyan → Indigo | `#0891B2 → #4F46E5` | Спортивный, активный |
| 4 | Emerald → Teal | `#059669 → #0891B2` | Природа, поход |
| 5 | Amber → Rose | `#D97706 → #E11D48` | Тёплый, юбилей |
| 6 | Business | `#0F172A → #1E3A8A` | Деловая встреча |

Все градиенты идут под одним углом (135°), палитра согласована — нет «случайного» зелёного рядом с красным.

## Компоненты

### Кнопки

- `border-radius: 10px` — чуть crisp-ее, чем `lg` (12px)
- Единственный primary: Indigo `#4F46E5`
- Hover: поднимается на 1px + мягкая indigo-тень
- Active: возвращается, темнеет до `#3730A3`

### Поля ввода

- Focus ring: `0 0 0 3px rgba(79, 70, 229, 0.12)` — indigo, не purple
- Border hover: `#A5B4FC` (indigo-300) вместо gray.300

### Карточки

- Border: `1px solid rgba(15, 23, 42, 0.07)` — slate вместо black
- Полоска вверху: `4px` вместо 3px — чуть заметнее
- Hover: `-2px` translate + expanded shadow

### Статусы участников

| Статус | Цвет текста | Фон |
|--------|------------|-----|
| Подтверждён | `#059669` | `#D1FAE5` |
| Приглашён | `#D97706` | `#FEF3C7` |
| Отказался | `#DC2626` | `#FEE2E2` |

### Навигация

Стеклянный blur остаётся — это красиво и современно:
```
backdropFilter: blur(20px) saturate(1.8)
bg: rgba(255, 255, 255, 0.85)
```

---

## Декоративные элементы

**Страницы login/register** — блобы обновлены под новый бренд:
- Было: `radial(brand.200, transparent)` + `#818CF8`
- Стало: `radial(#C7D2FE, transparent)` + `radial(#A5B4FC, transparent)` — мягче, harmonized

---

## Итог

Дизайн остаётся узнаваемым, но становится:
- **Самобытным** — уникальный Indigo вместо generic purple
- **Тёплым** — off-white фон, slate типографика
- **Согласованным** — один бренд-цвет везде
- **Современным** — refined gradients, crisp corners, subtle micro-interactions
